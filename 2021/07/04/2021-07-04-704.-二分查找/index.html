int mid = ((right - left) >> 1) + left; >>1 在这里这是什么意思

右移一位，相当于除以2，例如5的2进制表示是0101，5 >> 1以后为0010等于2，正好是5/2 = 2。如果是5 >> 2，则表示除以4，0101 右移两位之后变成 0001，正好为5/4 = 1。总的来说就是 >> 符号表示n除以2的k次方，k表示位移步数。同理<<表示左移，相当于乘以2的k次方。
